Implementace & úkoly
🌡 Loxone — test připojení

Volá Cloud Function getLoxoneTemp (region europe-west1), která z Miniserveru přes veřejnou IP přečte hodnotu čidla Tank 2 - skutečnost. Tlačítko spustí jeden test — výsledek (úspěch nebo přesný error) se zobrazí níže.

Klikni na tlačítko výše pro spuštění testu.
💡 eWeLink — spínání zásuvky

Cloud sync s eWeLink přes Firebase Cloud Function (region europe-west1). Po autorizaci a nastavení Device ID se v hlavičce zobrazí tlačítko pro rychlé spínání. Polling stavu každých 5 min (šetří API limit).
